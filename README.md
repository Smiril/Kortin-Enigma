## Instalation

```
cd Kortin-Enigma/bin
tar -xvzf enigma.tar.gz
sudo cp ./enigma /usr/local/bin/enigma
sudo mkdir -p /usr/local/share/enigma/
sudo cp *.txt /usr/local/share/enigma/

```
